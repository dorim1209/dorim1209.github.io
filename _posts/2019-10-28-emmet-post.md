Emmet jsx (tab키 누르면 자동 완성됨)

```
VS Code에서
Settings - Workspace Settings -  Extensions - Emmet

"Edit in settings.json" 파일을 찾아 아래 내용을 추가해준다.

   "emmet.includeLanguages": {
      "javascript": "javascriptreact"
   }
}
```
